# Sample Data

The semantic search example uses Nike's 2023 10-K PDF, the same sample document
used in the LangChain semantic search tutorial.

Download it into this folder as:

```text
data/nke-10k-2023.pdf
```

The PDF is intentionally not committed to the repo. Keeping large data files out
of Git makes the project easier to clone and keeps the examples focused on code.

If you use a different PDF, update `PDF_PATH` in `examples/semantic_search.py`.
